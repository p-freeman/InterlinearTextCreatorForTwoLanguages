# Text Splitter

The Text Splitter is a simple Python script with a graphical user interface (GUI) for splitting text into individual words while retaining empty lines.

## Usage

1. Run the script.
2. Enter or paste the text into the input textarea on the left.
3. Click the "Split Text" button to split the text.
4. The output with each word on a separate line will be displayed in the output textarea on the right.

## Features

- Graphical User Interface (GUI): Provides a user-friendly interface for text splitting.
- Retains Empty Lines: Preserves empty lines in the input text when splitting.
- Dynamic Textareas: Textareas automatically adjust their size to fit the content and expand when the window is resized.

## Requirements

- Python 3
- Tkinter library (usually included with Python installation)

## How to Run

1. Make sure you have Python installed on your system.
2. Run the script using a Python interpreter.
3. The GUI window will open, allowing you to split text interactively.

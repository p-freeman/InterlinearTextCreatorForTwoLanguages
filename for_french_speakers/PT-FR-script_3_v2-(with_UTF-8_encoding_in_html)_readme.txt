# Générateur de Texte Interlinéaire

Le Générateur de Texte Interlinéaire est un script Python permettant de créer des textes interlinéaires pour l'apprentissage des langues. Cette version modifiée permet la création de textes interlinéaires avec plusieurs paragraphes dans la zone de texte en portugais.

## Utilisation

1. **Exécution du Script**: Lancez le script pour ouvrir l'interface graphique utilisateur (GUI).
2. **Entrée des Détails du Projet**: Entrez un titre et un nom de fichier pour votre projet dans les champs respectifs.
3. **Entrée du Texte en Portugais**: Dans la zone de texte "Mots en Portugais", saisissez un mot par ligne. Utilisez une ligne vide pour indiquer un nouveau paragraphe.
4. **Entrée du Texte en Français**: Saisissez les mots français correspondants dans la zone de texte "Mots en Français". Chaque ligne devrait correspondre au texte en portugais.
5. **Paramétrage des Options**: Facultativement, définissez la distance verticale (en pixels) entre les paires interlinéaires et la taille de la police.
6. **Génération du Texte Interlinéaire**: Cliquez sur le bouton "Créer un Texte Interlinéaire" pour générer et enregistrer le texte interlinéaire.
7. **Ouverture d'un Projet Existant**: Alternativement, cliquez sur "Ouvrir un Projet Existant" pour charger et éditer un projet précédemment créé.

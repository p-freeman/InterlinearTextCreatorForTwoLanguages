# Diviseur de Texte

Le Diviseur de Texte est un script Python simple avec une interface graphique utilisateur (GUI) permettant de diviser un texte en mots individuels tout en conservant les lignes vides.

## Utilisation

1. Exécutez le script.
2. Saisissez ou collez le texte dans la zone de texte d'entrée à gauche.
3. Cliquez sur le bouton "Diviser le Texte" pour diviser le texte.
4. Le résultat avec chaque mot sur une ligne séparée sera affiché dans la zone de texte de sortie à droite.

## Caractéristiques

- Interface Graphique Utilisateur (GUI) : Fournit une interface conviviale pour diviser le texte.
- Conservation des Lignes Vides : Préserve les lignes vides dans le texte d'entrée lors de la division.
- Textareas Dynamiques : Les zones de texte ajustent automatiquement leur taille pour s'adapter au contenu et se développent lorsque la fenêtre est redimensionnée.

## Prérequis

- Python 3
- Bibliothèque Tkinter (généralement incluse dans l'installation de Python)

## Comment Exécuter

1. Assurez-vous d'avoir Python installé sur votre système.
2. Exécutez le script à l'aide d'un interpréteur Python.
3. La fenêtre de l'interface graphique (GUI) s'ouvrira, vous permettant de diviser le texte de manière interactive.

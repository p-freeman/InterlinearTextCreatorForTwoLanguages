# Interlinear Text Generator

Der Interlinear Text Generator ist ein Python-Skript zum Erstellen von interlinearen Texten für das Sprachenlernen. Diese modifizierte Version ermöglicht das Erstellen von interlinearen Texten mit mehreren Absätzen im Bereich der englischen Eingabe.

## Verwendung

1. **Skript ausführen**: Führen Sie das Skript aus, um die grafische Benutzeroberfläche zu öffnen.
2. **Projektdetails eingeben**: Geben Sie einen Titel und einen Dateinamen für Ihr Projekt in die entsprechenden Felder ein.
3. **Englischen Text eingeben**: Geben Sie im Textbereich "Englische Wörter" ein Wort pro Zeile ein. Verwenden Sie eine leere Zeile, um einen neuen Absatz anzugeben.
4. **Deutschen Text eingeben**: Geben Sie die entsprechenden deutschen Wörter im Textbereich "Deutsche Wörter" ein. Jede Zeile sollte dem englischen Text entsprechen.
5. **Optionen festlegen**: Optional können Sie den vertikalen Abstand (in Pixeln) zwischen den interlinearen Paaren und die Schriftgröße festlegen.
6. **Interlinearen Text erstellen**: Klicken Sie auf die Schaltfläche "Interlinearen Text erstellen", um den interlinearen Text zu generieren und zu speichern.
7. **Bestehendes Projekt öffnen**: Alternativ können Sie auf "Bestehendes Projekt öffnen" klicken, um ein zuvor erstelltes Projekt zu laden und zu bearbeiten.

# Text-Teiler

Der Text-Teiler ist ein einfaches Python-Skript mit einer grafischen Benutzeroberfläche (GUI) zum Aufteilen von Text in einzelne Wörter unter Beibehaltung leerer Zeilen.

## Verwendung

1. Führen Sie das Skript aus.
2. Geben Sie den Text in das Eingabe-Textarea links ein oder fügen Sie ihn ein.
3. Klicken Sie auf die Schaltfläche "Text aufteilen", um den Text aufzuteilen.
4. Die Ausgabe mit jedem Wort in einer separaten Zeile wird im Ausgabe-Textarea rechts angezeigt.

## Funktionen

- Grafische Benutzeroberfläche (GUI): Bietet eine benutzerfreundliche Oberfläche für das Aufteilen von Text.
- Behält leere Zeilen bei: Erhält leere Zeilen im Eingabetext beim Aufteilen.
- Dynamische Textareas: Textareas passen ihre Größe automatisch an den Inhalt an und werden beim Ändern der Fenstergröße erweitert.

## Anforderungen

- Python 3
- Tkinter-Bibliothek (in der Regel mit der Python-Installation enthalten)

## Ausführung

1. Stellen Sie sicher, dass Python auf Ihrem System installiert ist.
2. Führen Sie das Skript mit einem Python-Interpreter aus.
3. Das GUI-Fenster wird geöffnet, das Ihnen ermöglicht, den Text interaktiv aufzuteilen.

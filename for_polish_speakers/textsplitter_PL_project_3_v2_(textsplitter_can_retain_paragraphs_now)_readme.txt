# Podział Tekstu

Podział Tekstu to prosty skrypt w języku Python z graficznym interfejsem użytkownika (GUI) do dzielenia tekstu na poszczególne słowa zachowując puste linie.

## Użycie

1. Uruchom skrypt.
2. Wpisz lub wklej tekst do obszaru tekstowego wejściowego po lewej stronie.
3. Kliknij przycisk "Podziel Tekst", aby podzielić tekst.
4. Wynik z każdym słowem w osobnej linii zostanie wyświetlony w obszarze tekstowym wyjściowym po prawej stronie.

## Funkcje

- Interfejs graficzny użytkownika (GUI): Zapewnia przyjazny interfejs dla dzielenia tekstu.
- Zachowuje Puste Linie: Zachowuje puste linie w tekście wejściowym podczas dzielenia.
- Dynamiczne Obszary Tekstowe: Obszary tekstowe automatycznie dostosowują swój rozmiar do zawartości i rozszerzają się podczas zmiany rozmiaru okna.

## Wymagania

- Python 3
- Biblioteka Tkinter (zwykle dołączona do instalacji Pythona)

## Jak Uruchomić

1. Upewnij się, że masz zainstalowanego Pythona na swoim systemie.
2. Uruchom skrypt za pomocą interpretera Pythona.
3. Okno GUI zostanie otwarte, umożliwiając interaktywne dzielenie tekstu.

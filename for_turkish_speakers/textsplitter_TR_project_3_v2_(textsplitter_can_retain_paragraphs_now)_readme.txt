# Metin Bölme

Metin Bölme, boş satırları koruyarak metni kelimelere ayıran basit bir Python betiği ve grafiksel kullanıcı arayüzü (GUI)'dür.

## Kullanım

1. Betiği çalıştırın.
2. Metni sol taraftaki metin alanına yazın veya yapıştırın.
3. Metni bölmek için "Metni Böl" düğmesine tıklayın.
4. Her kelimenin ayrı bir satırda olduğu sonuç sağ taraftaki metin alanında görüntülenecektir.

## Özellikler

- Grafiksel Kullanıcı Arayüzü (GUI): Metni bölmek için dostça bir arayüz sağlar.
- Boş Satırları Korur: Metni bölerken giriş metnindeki boş satırları korur.
- Dinamik Metin Alanları: Metin alanları içeriğe göre otomatik olarak boyutlanır ve pencere boyutu değiştirildiğinde genişler.

## Gereksinimler

- Python 3
- Tkinter Kütüphanesi (Genellikle Python kurulumuyla birlikte gelir)

## Nasıl Çalıştırılır

1. Sistemine Python kurulu olduğundan emin ol.
2. Betiği Python yorumlayıcısı ile çalıştır.
3. GUI penceresi açılacak ve etkileşimli olarak metin bölme imkanı sağlayacaktır.
